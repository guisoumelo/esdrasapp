Política de Privacidade — Esdras

Última atualização: 14 de julho de 2026

O aplicativo Esdras foi desenvolvido como um recurso educativo sobre as 28 Crenças Fundamentais da Igreja Adventista do Sétimo Dia. Esta página informa sobre nossa política em relação à coleta, uso e divulgação de dados pessoais.

1. Coleta de Dados

O aplicativo Esdras não coleta nenhum dado pessoal dos usuários. O app funciona 100% offline, sem conexão com servidores externos, sem cadastro de conta e sem rastreamento de qualquer tipo.

2. Dados Armazenados Localmente

O aplicativo armazena localmente no dispositivo do usuário apenas informações de progresso do quiz (pontuações e histórico de respostas), exclusivamente para fins de experiência do usuário. Esses dados nunca são transmitidos para fora do dispositivo.

3. Serviços de Terceiros

O aplicativo Esdras não utiliza nenhum serviço de terceiros, SDKs de análise, publicidade ou rastreamento.

4. Crianças

O aplicativo não coleta dados de nenhum usuário, incluindo menores de 13 anos. É seguro para todas as idades.

5. Alterações nesta Política

Esta política pode ser atualizada periodicamente. Recomendamos que os usuários revisem esta página ocasionalmente para verificar eventuais mudanças.

6. Contato
